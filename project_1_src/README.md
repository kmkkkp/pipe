# 시소보 Project_1
## 리다이렉트 활용 및 입출력을 파일로 받기
### 실행 파일 생성
1. P 실행파일
<pre>
<code>
gcc P.c -o P
</code>
</pre>

2. project_1 실행파일
<pre>
<code>
gcc project_1.c -o project_1
</code>
</pre>

### 실행
1. P 실행파일
<pre>
<code>
./P
</code>
</pre>
입력, 출력을 확인할 수 있음

2. project_1 파일 실행 전 수행함(실행되거나 오픈되는 파일들의 디렉토리는 같게 둔다)
<pre>
<code>
vi A
</code>
</pre>
A의 내용을 project_1에서 입력으로 활용할 것이므로, A 파일에 내용을 작성한다.

3. project_1 파일 실행
<pre>
<code>
./project_1
</code>
</pre>
