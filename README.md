# 클라이언트-서버 모델을 이용한 concurrent 파일 서버 구현
## executable file 생성
1. server
<pre>
<code>
gcc server.c -o server
</code>
</pre>

2. client
<pre>
<code>
gcc client.c -o client
</code>
</pre>

## executable file 실행
>두 개의 터미널을 연 후에 server file을 먼저 실행해야 한다.
<pre>
<code>
./server
./client
</code>
</pre>
